My Projects

→ The Nasa Bot

A Twitter Bot using the NASA & Twitter API written in python.
  • Uses NASA's Picture of the day API to post daily updates.
  • Uses NASA's Image & Video Library API to reply to user requests with an image.
  • Uses python's requests & beautifulsoup libraries to scrape NASA's News website and post new updates to Twitter.

→ Mini Home Server

A Home Server using RaspberryPi 4 Single Board Computer.
  • Linux System Administration and Management
  • Using the Docker platform to deploy various apps and services to be used within the home network
  • Setting up Internal Domain Name and Proxy for all running services
  • Configuring Reverse VPN to expose certain services to the public internet
  • Setting up NAS service for centralized file storage and backups.
